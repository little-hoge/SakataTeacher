##第二回 

#####抽象クラスとは
- 抽象メソッドは継承先で実装  
- 実装済みメソッドも記述可能  
- 継承先は一意  

#####インタフェースとは
- メンバ変数はfinal public static(定数)  
- メソッドはpublic abstract(抽象メソッド)  
- 継承先は複数可(ミックスイン)  
- メンバ変数は一意
