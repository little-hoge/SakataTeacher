#坂田先生のJava講習メモ

アクセス修飾子
・何も付けない場合privateではなく同じパッケージ内のみ参照可能

汎化と特化(is-a関係)と集約（part-ofまたはhas-a）
・親クラスにまとめることを「汎化」
・子クラスに付け加えることを「特化」

・継承関係 …サブクラス is a スーパクラス。
・包含関係 …全体クラス has a 部分クラス。


抽象クラスとは
・抽象メソッドは継承先で実装
・実装済みメソッドも記述可能
・継承先は一意


インタフェースとは
・メンバ変数はfinal public static(定数)
・メソッドはpublic abstract(抽象メソッド)
・継承先は複数可(ミックスイン)
・メンバ変数は一意
